# MakeTeamUpAppFrontEnd
FrontEnd
校园组队APP（Team-Up App Android）
一.	项目介绍
一款基于Android的校园组队信息发布的app，可实现大学校园寻找志趣相投的人的功能。
基本功能为参与组队，具体包括如下模块：
1)	登录注册模块
2)	组队分类模块
3)	组队搜索模块
4)	组队推荐模块
二.	相关技术
1.	Okhttp + json数据传输和解析
2.	Material Design UI设计
3.	百度自然语言处理技术
a)	中文分词技术
b)	文本审核技术
c)	短文本相似度技术
