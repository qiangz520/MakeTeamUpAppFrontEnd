# TeamUpAppFrontEnd
## 校园组队APP（Team-Up App Android）
### 项目介绍
  一款基于Android的校园组队信息发布的app，可实现大学校园寻找志趣相投的人的功能。基本功能为参与组队，具体包括如下模块：
* 登录注册模块
* 组队分类模块
* 组队搜索模块
* 组队推荐模块
### 相关技术
* Okhttp + json数据传输和解析
* Glide图片加载框架
* Material Design UI设计：RecyclerView、DrawerLayout、SwipeRefresh、FloatingActionButton
* 自然语言处理技术
  1）中文分词技术
  2）文本审核技术
  3）短文本相似度技术
### 功能模块
* 登录注册
  
* 组队分类
  
* 发布组队
  
* 组队搜索
  
* 组队推荐

* 个人信息
